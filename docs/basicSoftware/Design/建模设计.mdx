## Crow's Foot Notation


* 分为零，一和多的三种关系
* 一个方向会有两种关系AB，表示最小A关系最大B关系

| 符号  | 含义  |
|---|---|
| o   | 零  |
| \| | 一  |
| \{  | 多  |

``` mermaid
---
title: 示例
---
erDiagram

    "老师" }|--|| "课程" : "每个老师只教一门课程，一门课程会有多个教师"
    "学生" ||--|| "座位" : "一对一"
    "顾客" ||--o{ "订单" : "一个顾客零份到有多份订单，每份订单只属于一个顾客"

```
